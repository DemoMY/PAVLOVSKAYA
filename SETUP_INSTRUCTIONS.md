# Инструкция по настройке нейросети в приложении

## Настройка API-ключа для Google Gemini

1. Получите API-ключ от Google Gemini:
   - Перейдите на сайт [Google AI Studio](https://aistudio.google.com/)
   - Создайте аккаунт или войдите в существующий
   - Создайте новый API-ключ

2. Создайте или отредактируйте файл `.env` в корне проекта:

```
VITE_GEMINI_API_KEY=ваш_реальный_API_ключ_здесь
```

3. Установите необходимые зависимости:

```bash
npm install
```

4. Запустите приложение:

```bash
npm run dev
```

## Как получить API-ключ

1. Перейдите на [Google AI Studio](https://aistudio.google.com/)
2. Нажмите "Get API Key"
3. Выберите или создайте проект
4. Скопируйте API-ключ и вставьте в файл `.env`

## Важные замечания

- Не публикуйте API-ключи в открытых репозиториях
- Файл `.env` уже добавлен в `.gitignore`, чтобы случайно не опубликовать ключи
- При использовании API-ключа в браузере, ваш ключ может быть виден другим пользователям

## Альтернативное решение

Если у вас нет API-ключа, нейросеть будет возвращать сообщение об ошибке. Вы можете реализовать альтернативный вариант работы (например, с локальной моделью или фиктивными ответами) в файле `/services/geminiService.ts`.

## Тестирование работы

После настройки API-ключа, откройте чат-модуль в приложении и отправьте сообщение. Если все настроено правильно, вы должны получить ответ от нейросети.
